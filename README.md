## 🌐 SEO & Performance

Followed for best results:

- ✔️ **Meta description:** 150–160 chars, keywords included
- ✔️ **Canonical tag:** Self-referencing
- ✔️ **Semantic HTML:** Proper heading structure
- ✔️ **Mobile-first:** Responsive design, viewport-fit
- ✔️ **Structured Data-ready:** Extend with JSON-LD if needed
- ✔️ **Core Web Vitals:** Minimal CLS, fast CSS, no heavy JS
- ✔️ **GA4 + GTM:** Analytics & events ready
